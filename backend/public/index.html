1 Install AltStore

AltStore lets you install apps outside the App Store. You only need to do this once.

On Mac:

  1. Download AltServer from altstore.io
  2. Open AltServer - it will appear in your menu bar
  3. Connect your iPhone to your Mac with a cable
  4. Click the AltServer icon ‚Üí Install AltStore ‚Üí [Your iPhone]
  5. Enter your Apple ID when prompted

On Windows:

  1. Download AltServer from altstore.io
  2. Install iTunes and iCloud from Apple's website (not the Microsoft Store)
  3. Run AltServer
  4. Connect your iPhone with a cable
  5. Click the AltServer icon ‚Üí Install AltStore ‚Üí [Your iPhone]
  6. Enter your Apple ID when prompted

2 Add the cTikTok Source

  1. Open AltStore on your iPhone
  2. Go to the Sources tab
  3. Tap the + button
  4. Enter this URL:
https://ctiktok.roshanc.com/altstore/source.json

Or tap this button on your iPhone:

Add to AltStore

3 Install cTikTok

  1. In AltStore, go to Browse
  2. Find cTikTok in the list
  3. Tap GET to install
  4. The app will appear on your home screen!

4 Create Your Account

  1. Open cTikTok
  2. Tap Register
  3. Choose a username and password
  4. Start sharing videos!
